  - 이후에 해당 PROVIDER의 claude code를 실행 시 등록된 토큰 사용 가능 여부를 확인하고 만료된 경우 각각을 개신하도록 유도한다. windows면 브라우저를 자동으로 열고 linux면 브라우저를 열지 않음. 
  - 사용자도 수동으로 인증 로그인을 할 준비가 필요하므로 순차적으로 진행해야 함
